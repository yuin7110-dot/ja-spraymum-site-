# JAひまわりスプレーマム部会サイト（保守・改善）エージェント指示

あなたは「JAひまわりスプレーマム部会」ホームページの保守・改善を担当するコーディングエージェントです。

## 目的
豊川市のスプレーマム産地として、品質・安定供給・共選共販・就農支援の魅力が正確に伝わるサイトにする。

## 基本方針
- 既存のHTML/CSS/JavaScript構成を尊重し、大きな作り替えは避ける。
- 日本語を主にし、見出しやナビは既存どおり英語併記を維持する。
- 品種名、出荷時期、規格、連絡先、生産者数などは「確認済み情報だけ」を書く。
- 不明な情報は「未登録」「確認中」「要確認」と明記し、推測で埋めない。

## 主な作業対象
- `index.html`：産地紹介と導線の改善
- `varieties.html` と `varieties/`：品種追加・更新
- `business.html`：花屋・市場向け情報の改善
- `consumer.html`：一般消費者向け情報の改善
- `farmer.html`：就農希望者向け情報の改善

## 品種追加の手順
- 追加手順は `varieties/ADDING_VARIETIES.md` に従う（画像 → `js/varieties.js` → 詳細ページ）。

## 完了チェック（必ず）
- スマホ表示崩れがない（幅 375px 前後で確認）
- 画像 `alt` が適切（品種名・花色など）
- 見出し階層（`h1`→`h2`→`h3`）が破綻していない
- リンク切れ・画像パス切れがない
- 文字化け（エンコーディング不一致）が起きていない
- `varieties.html` の検索・絞り込みが意図どおり動く

## 禁止事項
- 架空の品種・数値・市場評価を書かない。
- 電話番号・住所・出荷規格は勝手に変更しない（要確認のうえで変更する）。
- GitHub Pages / Vercel 等のデプロイ設定は、合意なく変更しない。
